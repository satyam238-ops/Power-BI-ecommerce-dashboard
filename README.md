# Power-BI-ecommerce-dashboard
📊 E-Commerce Sales Dashboard (Power BI)

🔹 Overview

This project analyzes e-commerce sales, customer behavior, and operational performance using Power BI.

🔹 Tools Used

- Power BI
- Power Query
- DAX

🔹 Key Features

- KPI metrics (Sales, Profit, Orders, Margin)
- Regional analysis
- Customer insights (Top customers)
- Delivery performance tracking

🔹 Key Insights

- West region has highest sales but lower profit margin
- Top customers contribute major revenue
- Discounts reduce profitability
- Most deliveries completed within 3–5 days
